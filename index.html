Foundations and Milestones

The journey of the internet began in the late 1960s with the creation of ARPANET, funded by the U.S. Department of Defense. It was the first network to implement the TCP/IP protocol suite.

This early network demonstrated that packet switching (breaking data into small chunks) was more efficient than traditional circuit switching. Many historians also note that DNS (Domain Name System) was a crucial turning point in making the web accessible.

The Birth of the World Wide Web

While often used interchangeably, the Internet and the World Wide Web are different. Below are key components introduced by Tim Berners-Lee in 1989:

  • HTML (HyperText Markup Language)
  • HTTP (HyperText Transfer Protocol)
  • The first web browser (WorldWideWeb)
"The Web is more a social creation than a technical one. I designed it for a social effect — to help people work together — and not as a technical toy." — Tim Berners-Lee

Early web pages were written in basic HTML, similar to the following code snippet often used for demonstration: 

    <!DOCTYPE html>
    <html>
    <body>
    <h1>Welcome to the Web</h1>
    </body>
    </html>
