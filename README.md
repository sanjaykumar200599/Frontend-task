# HTML5 Design Task

This branch contains the implementation of the given design using **HTML5 only**, as per the assignment requirements.

## Overview
The page is developed without using any CSS or JavaScript.  
The focus is on demonstrating correct HTML structure, semantics, and accessibility based on the provided design reference.

## HTML Concepts Covered
- HTML5 document structure and semantic elements
- Headings, paragraphs, lists, and hyperlinks
- Forms with labels, input types, validation, fieldsets, and legends
- Media elements (image, audio, video) with accessibility attributes
- Tables for structured data representation
- Global attributes and meta tags
- SEO-related meta information
- HTML entities and symbols
- Accessibility best practices

## Technologies Used
- HTML5

## Notes
This branch intentionally excludes CSS and JavaScript.  
The objective is to demonstrate a strong foundation in semantic HTML and content structure.
